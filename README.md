Смешанный плейлист iptv Руские каналы тв  в формате m3u.

Для Андроид ТВ.
Работают на проигрывателе.
Wuffy Player (версия : 3.5.7 arm64-v8a) apk.
Wiseplay (версия: 8.1.34-tv) apk.

(На других проигрывателях может не работать).

Программу для IPTV можно использовать любую:  Lazy IPTV, IPTV, или любую другую для удобства.
